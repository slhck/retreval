TestQueryResult

Some basic unit tests for QueryResult Unranked results include 4 documents of 10, which are all retrieved. The ranked results are evaluated from this table: 

     Index Relevant  Precision   Recall        Document ID
     1     [X]       1.000        0.250        doc1
     2     [X]       1.000        0.500        doc2
     3     [ ]       0.667        0.500        doc5
     4     [X]       0.750        0.750        doc3
     5     [ ]       0.600        0.750        doc6
     6     [X]       0.667        1.000        doc4
     7     [ ]       0.571        1.000        doc7
     8     [ ]       0.500        1.000        doc8
     9     [ ]       0.444        1.000        doc9
     10    [ ]       0.400        1.000        doc10
